Position

Значения свойства position: static - по умолчанию relative - относительное позиционирование. Мы можем двигать элементы с помощью top left right bottom absolute - абсолютное позиционирование относительно родителя. fixed - фиксированное позиционирование относительно обасти просмотра. sticky - прилипающее позиционирование.
